    Ép kiểu hẹp: Ép kiểu từ dữ liệu lớn thành bé. Double => Float => Long => Int. Có thể làm mất dữ liệu

    8. Toán tử logic, Toán tử tiền tố, hậu tố (prefix, postfix) - lập trình C++ căn bản

    1. Toán tử logic && || !

    2. Toán tử tiền tố (prefix),hậu tố (postfix)

    prefix: dấu viết trước biến

    ++x Tăng giá trị x lên 1. Tương đương x=x+1

    --y Giảm giá tri y đi 1. Tương đương y=y-1

    postfix: dấu viết sau biến

    x++ Tăng giá trị x lên 1. Tương đương x=x+1

    y-- Giảm giá tri y đi 1. Tương đương y=y-1

    Trường hợp phép tính đơn lẻ thì tuân theo quy tắc như trên. Tuy nhiên với những phép gán, phép tính phức tạp thì tuân theo quy tắc bên dưới

    Step 1: Prefix

    Step 2: Các phép tính còn lại

    Step 3: Gán giá trị cho biến ở bên trái dấu bằng

    Step 4: Postfix

    E.g: let z =x++ - ++y + 1 với x=2, y=4

    Step 1: y = y+1

    Step 2,3: z = x - (y+1) + 1 = 2 - (4+1) + 1 = -2

    Step 4: x = x+1

    Kết quả: x=3, y=5, z=2-5+1=-2 (không phải bằng 3-5+1=-1)

    #include <iostream>
            #include <climits>
            using namespace std;
            int main()
            {
              cout << "Hello World without std:: !\n";
              // 1. Toán tử logic && || !
              int i = 10;
              cout << (i > 0 && i < 10) << endl;
              cout << (i > 0 || i < 10) << endl;
              cout << (!i) << endl;
            
              // 2. Toán tử tiền tố (prefix),hậu tố (postfix)
              int a = 5;
              cout << "a = " << a << endl;
              a++;
              cout << "a++ = " << a << endl;
              int b = 10;
              cout << "b = " << b << endl;
              b--;
              cout << "b-- = " << b << endl;
            
              int x = 1;
              int y = 2;
              int z = x++ - ++y + 1; // x = 2, y = 3, z = -1
              cout << "Du doan x = 2, y = 3, z = -1" << endl;
              cout << "x = " << x << endl;
              cout << "y = " << y << endl;
              cout << "z = " << z << endl;
